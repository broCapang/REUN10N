## Question:
NICC recieved a mysterious email with an executable file that does nothing. Can you figure out what this executable does?


## Solution:
1. dissamble the elf
2. read the opcode
3. decode any hex declaration (word, dword, qword, byte)

**Flag:** `NICC{uhH_fl@g_i_ThInk}`
